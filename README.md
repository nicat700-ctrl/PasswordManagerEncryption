# Password Manager Encryption 🔐

A secure password manager built with Python that stores passwords using encryption.

## Features

- Add and store passwords securely
- Encrypt passwords before saving
- Decrypt passwords when needed
- Generate and manage encryption keys
- Protect sensitive data using cryptography

## Technologies

- Python
- Cryptography
- Fernet Encryption
- File Handling

## Security

Passwords are encrypted before being stored.

Sensitive files such as:

- key.key
- passwords.txt

are ignored using .gitignore for security.
